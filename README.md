# To-Do List App

A simple and user-friendly To-Do List application that helps users organize tasks, track progress, and improve productivity through efficient task management.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Simple and clean user interface
* Easy task management

## Technologies Used

* HTML
* CSS
* JavaScript

## Description

This project is designed to make daily task management easier by allowing users to create, manage, and complete their tasks in an organized way.
